# spec-first-doc

`spec-first-doc` 是 `spec-first` 的文档工程仓库，用于沉淀对外文章、业界学习材料、内容运营规划和可复用写作资产。

## 目录导航

- [文章系列](./文章系列/README.md)：微信公众号文章、分享讲稿、发布台账和运营规划。
- [业界学习](./业界学习/README.md)：从 `spec-first` 主仓旧 `docs/09-业界借鉴/` 迁入的外部文章、对标研究、机制分析和能力映射资料。

## 推荐阅读路径

1. 先读 [业界学习/01-外部文章](./业界学习/01-外部文章/README.md)，建立 AI Coding、Spec、Harness Engineering、知识基座和度量体系的外部参考背景。
2. 再读 [业界学习/02-对标研究](./业界学习/02-对标研究/README.md)，理解 `spec-first` 与 Harness、CRG、Compound Engineering、Agent OS 等方向的差距和演进线索。
3. 最后回到 [文章系列](./文章系列/README.md)，把学习材料转化为面向读者的文章、路线图和发布内容。

## 维护约定

- 新增或移动文档时，同步更新对应目录的 `README.md`。
- 文章正文和同名 `.assets/` 资源目录保持相邻，优先使用相对链接。
- 任何项目文档变更都必须同步更新根目录 `CHANGELOG.md`。
