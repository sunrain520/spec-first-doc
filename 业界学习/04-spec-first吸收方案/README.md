# spec-first 吸收方案

本目录保存“如何把业界资料转成 spec-first 能力”的二次加工材料。这里不是原始竞品资料，而是能力映射、路线图、差距审计和可落地改造建议。

## 分组

- [能力映射](./能力映射/)：核心链路、工具矩阵、能力缺口和同步审计。
- [skill-agent 映射](./skill-agent映射/)：按 skill / agent 记录能力对齐、增强方案和演进建议。
- [路线图与差距](./路线图与差距/README.md)：承接跨主题的差距判断、优先级和路线图。
- [可落地改造建议](./可落地改造建议/README.md)：承接可直接进入 plan/work 的实现建议。
