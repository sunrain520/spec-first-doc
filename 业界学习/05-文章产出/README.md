# 文章产出

本目录管理从业界资料到可发布文章的工作流。

## 流转目录

- [00-选题池](./00-选题池/README.md)：候选文章主题。
- [01-大纲](./01-大纲/README.md)：已选题但未写正文。
- [02-草稿](./02-草稿/README.md)：正在写作或待评审正文。
- [03-待发布](./03-待发布/README.md)：已完成、等待发布排期。
- [04-已发布](./04-已发布/README.md)：发布后的最终稿和发布信息。
- [assets](./assets/README.md)：文章配图、封面和发布素材。

## 使用约定

- 资料引用从 `00-阅读地图/资料到文章映射.md` 开始登记。
- 大纲和草稿不要直接放到资料库目录。
- 发布后把最终稿移动到 `04-已发布/`，草稿区只保留仍在工作的版本。
