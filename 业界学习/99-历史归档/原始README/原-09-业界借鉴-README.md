## 目录约定

- 每篇文章使用一个 `.md` 文件保存正文。
- 每篇文章配套一个同名 `.assets/` 目录保存本地图片。
- Markdown 中的图片链接统一使用相对路径，便于离线阅读和仓库内跳转。

## 后续维护

- 新增文章时，保持“正文文件 + 同名资源目录 + 本索引登记”三件套一致。
- 若后续对文章做排版精修，尽量只改 Markdown 结构，不改原文内容。
