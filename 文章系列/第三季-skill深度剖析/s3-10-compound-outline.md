# Spec-First：你解决过的问题，为什么下次还要重新解决一遍

**状态：** 大纲
**内容类型：** 机制篇

## 核心论点

compound 不是写文档，而是在上下文最新鲜时把可复用经验固化下来。compound-refresh 用六种处理动作维护知识质量：Keep / Update / Consolidate / Replace / Delete / Stale-mark。删除而不是归档。
