# 运营规划

spec-first 微信公众号系列的规划与路线图目录。所有文档的编号、定位与阅读顺序都在这里。

## 怎么读

- **第一次了解整个系列** → 先读 `006-顶层总规划.md`，它统摄三季（设计思想/项目实操/细节设计）、含编号对账表和演化说明。
- **想知道先发什么、怎么拉新** → 读 `009-运营策略与发布日历.md`，发布节奏、钩子流、BIP 排期、增长指标。
- **想动手用 spec-first** → 读 `spec-first-使用说明.md`，一个贯穿案例从安装串到经验沉淀。
- **要写某一季的文章** → 第一季读 004，第二季实操读 008，第三季细节读 005（skill 编号）/ 007（进阶专题）。
